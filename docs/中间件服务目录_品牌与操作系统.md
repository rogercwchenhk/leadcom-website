# 中间件服务目录 — 品牌与操作系统

> 版本：v1.0 | 更新日期：2026-07-01
> 适用范围：第三方IT服务商中间件运维、部署、迁移、信创替代项目

## 一、中间件品牌与操作系统总览

### 1.1 分类体系

| 分类 | 说明 |
|------|------|
| **应用服务器中间件** | Java EE / Jakarta EE 应用服务器，部署和管理企业Java应用 |
| **Web服务器/负载均衡** | HTTP反向代理、静态资源服务、流量分发 |
| **消息中间件** | 异步消息传递、事件驱动、解耦系统通信 |
| **交易中间件** | 分布式事务处理（TP Monitor） |
| **集成中间件/ESB** | 企业服务总线、SOA集成、API编排 |
| **缓存中间件** | 内存级数据缓存、会话管理、高速KV存储 |
| **API网关** | API管理、流量控制、安全认证、服务治理 |

> 注：容器平台（Docker/K8s/OpenShift等）已独立至《云基础设施服务目录》

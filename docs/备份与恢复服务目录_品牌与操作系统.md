# 备份与恢复服务目录

## 一、服务概述

备份与恢复是数据安全保障的核心，涵盖数据备份、灾难恢复、业务连续性管理。本目录覆盖企业级备份软件、备份一体机、容灾与高可用产品。

## 四、选型指南

### 4.1 按企业规模选型

| 企业规模 | 数据量 | 推荐产品 | 理由 |
|---------|--------|---------|------|
| **小微企业** | < 10TB | Veeam Community/Backup Exec、鼎甲AnBay | 易部署、低成本、足够功能 |
| **中小企业** | 10-100TB | Veeam B&R Standard、云祺Vinchin、爱数AnyBackup | 性价比高、支持国产环境 |
| **大型企业** | 100TB-1PB | Commvault、Veritas NetBackup、华为OceanProtect、鼎甲DBackup | 功能全面、高扩展、强生态 |
| **超大型/集团** | > 1PB | Commvault、Veritas NetBackup、华为OceanProtect X8000/X9000 | 分布式架构、PB级管理 |

### 4.2 按信创需求选型

| 信创要求 | 推荐产品 | 支持架构 |
|---------|---------|---------|
| **全栈信创（芯片+OS+DB）** | 华为OceanProtect/DPA、鼎甲DBackup | 鲲鹏/飞腾/海光/龙芯/申威 + 麒麟/统信 + 达梦/人大金仓/GaussDB |
| **部分信创（OS+DB）** | 爱数AnyBackup、英方i2Backup、云祺Vinchin | x86/ARM + 麒麟/统信 + 国产数据库 |
| **无信创要求** | Veeam、Commvault、Veritas | x86 + Windows/RHEL + Oracle/SQL Server |

### 4.3 按场景选型

| 场景 | 核心需求 | 推荐产品 |
|------|---------|---------|
| **VMware虚拟化保护** | 无代理备份、即时恢复 | Veeam B&R、Commvault、鼎甲DBackup |
| **国产化替代** | 适配国产CPU/OS/DB | 华为OceanProtect、鼎甲DBackup、云祺Vinchin |
| **VMware替代迁移** | 同时支持VMware+国产虚拟化 | 华为OceanProtect、鼎甲DBackup、云祺Vinchin |
| **数据库实时保护** | CDP、日志级RPO | 英方i2CDP、爱数AnyBackup |
| **异地灾备** | 远程复制、两地三中心 | Commvault、Veritas NetBackup、华为OceanProtect、鼎甲DBackup |
| **云原生/K8s** | 容器数据保护 | 鼎甲DBackup（K8s）、华为OceanProtect、Commvault |
| **防勒索** | 不可变备份、Air Gap | Veeam、华为OceanProtect、Commvault |
| **分支机构/边缘** | 轻量化、统一管理 | Veeam、鼎甲AnBay、云祺Vinchin |

### 4.4 按成本选型

| 方案 | 初始投入 | 长期成本 | 适用 |
|------|---------|---------|------|
| **开源（Bacula/Bareos）** | 零（软件免费） | 中（需专业运维） | 有Linux运维团队 |
| **Veeam** | 中（按CPU Socket） | 低（功能集成度高） | VMware/Hyper-V环境 |
| **华为OceanProtect一体机** | 高（软硬一体） | 低（开机即用、重删率高） | 大中型数据中心 |
| **鼎甲DBackup** | 中 | 低 | 信创环境、多虚拟化 |
| **Commvault/Veritas** | 高 | 中 | 大型企业、多云环境 |

### 4.5 选型决策树

```
是否需要信创适配（国产CPU/OS/DB）？
├─ 是 → 是否全栈信创（六架构全覆盖）？
│   ├─ 是 → 鼎甲DBackup（唯一六架构支持）
│   └─ 否 → 华为OceanProtect / 爱数AnyBackup / 云祺Vinchin
└─ 否 → 虚拟化平台？
    ├─ VMware为主 → Veeam B&R（功能最强）
    ├─ 混合虚拟化 → Commvault / 鼎甲DBackup / 云祺Vinchin
    └─ 纯物理机 → Veritas NetBackup / Commvault

是否需要备份一体机（软硬一体）？
├─ 是 → 华为OceanProtect（性能最强）/ 云祺Vinchin（性价比）
└─ 否 → 纯软件部署 → Veeam / Commvault / 鼎甲DBackup

是否需要防勒索/不可变存储？
├─ 是 → Veeam + Immutability / 华为OceanProtect + WORM/AirGap
└─ 否 → 按其他条件选型
```

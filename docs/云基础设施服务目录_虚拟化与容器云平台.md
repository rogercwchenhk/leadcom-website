# 云基础设施服务目录 — 虚拟化与容器云平台

> 版本：v1.0 | 更新日期：2026-07-01
> 适用范围：第三方IT服务商虚拟化运维、容器平台部署、信创替代、超融合项目

---

## 一、云基础设施总览

### 1.1 分类体系

| 分类 | 说明 |
|------|------|
| **服务器虚拟化平台** | 基于Hypervisor的虚拟机管理，VMware替代主战场 |
| **超融合基础设施 (HCI)** | 计算+存储+网络融合的软件定义基础设施 |
| **容器运行时** | Docker/Podman等容器引擎 |
| **容器编排平台** | Kubernetes及企业级K8s发行版 |
| **云管平台 (CMP)** | 多云/混合云统一管理、资源编排、自服务门户 |

### 1.2 市场背景（2026）

- Broadcom收购VMware后取消永久授权、全面订阅制，大量企业面临成本数倍增长
- Gartner预测：到2028年70%的VMware客户将迁移50%的虚拟工作负载
- 国产虚拟化（华为FusionCompute、新华三CAS、深信服、ZStack）快速填补市场
- 容器平台成为云原生标准，K8s生态成熟
